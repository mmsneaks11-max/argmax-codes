# Compyoot

Multi-provider AI routing engine. Classifies task complexity and directs
work to the optimal model. Built solo by Clawd after a parallel dispatch
to five agents failed — one agent, one build, shipped.

## How it works

1. Task arrives.
2. Compyoot classifies complexity.
3. Router picks the cheapest model that can actually do the job.
4. Low-complexity work routes to free tiers (Gemini, Mistral Small, Groq
   Llama, Cerebras, NVIDIA Llama).
5. High-complexity work routes to Claude.

## Why it matters

Cost-of-compute is the moat for the whole Agent Deck stack. Compyoot is
how we defend it.
