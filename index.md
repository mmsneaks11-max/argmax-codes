**argmax** `(ˈɑːɡmæks)` — *n.* the argument that returns the maximum.
Given a set of options, return the best one. That's this directory.

---

## What this is

A flat, Markdown-first index of every property, agent, and client site in
the Agent Deck org. No JavaScript. No tracking. No cookie banners. Just
structured text any agent can `curl` and parse in one shot.

Every page on this site has a machine-readable frontmatter block and a
human-readable body. Agents: parse the frontmatter. Humans: read the prose.
